# Plateforme bancaire IA - MVP

Prototype de plateforme bancaire intelligente pour une agence, centré sur le scénario MVP :

1. Un conseiller saisit une transaction.
2. Le backend enregistre la transaction.
3. Le module IA calcule un score de risque.
4. Une alerte est créée si le score est élevé.
5. Le directeur consulte l'alerte dans le dashboard.

## Stack prévue

- Backend : FastAPI
- Base de données : PostgreSQL
- ORM : SQLAlchemy
- Migrations : Alembic
- IA : scikit-learn
- Frontend : React
